server-testlib/io.spine.testing.server.blackbox

Package-level declarations

This package provides test utilities for implementing black box server testing. Such a tests would provide an ability to test complex systems without setting up the infrastructure.

One such black box example is for Bounded Context testing. It allows sending Commands and Events to the Bounded Context and then verifying their effect inside the Bounded Context.
