EmittedMessageSubject

abstract class EmittedMessageSubject<S : EmittedMessageSubject<S, T, M>?, T : Signal<out Any, out Any, out Any>?, M : SignalMessage?> : Subject

Abstract base for subjects checking messages, such as events or commands, generated by a Bounded Context under the test.

Parameters

<S>

the type of the concrete subject implementation (e.g., EventSubject or CommandSubject), which enables method chaining with the correct return type

<T>

the type of the outer objects of the checked messages, such as Command or Event

<M>

the type of emitted messages, such as CommandMessage or EventMessage
