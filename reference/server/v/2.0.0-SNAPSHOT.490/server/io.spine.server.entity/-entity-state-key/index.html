class EntityStateKey : GeneratedMessage, EntityStateKeyOrBuilder, ValidatableMessage
Identifies a stored state of an entity: the entity and the version
at which the state became current.

Serves as the record key of the entity state history, so writing the state
of the same entity at the same version again overwrites one record instead
of creating a duplicate.
