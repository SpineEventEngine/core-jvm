RecordBasedRepository

abstract class RecordBasedRepository<I, E : Entity<I, S>?, S : EntityState<I>?> : Repository<I, E> , QueryableRepository<I, S>

The base class for repositories that store entities as records.

Such a repository is backed by EntityRecordStorage. Entity states are stored as EntityRecords.

Parameters

<I>

the type of entity identifiers

<E>

the type of stored entities
