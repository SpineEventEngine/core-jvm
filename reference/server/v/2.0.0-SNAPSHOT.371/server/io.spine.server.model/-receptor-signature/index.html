ReceptorSignature

abstract class ReceptorSignature<R : Receptor<out Any, out Any, E, out Any>?, E : MessageEnvelope<out Any, out Any, out Any>?> : WithLogging

Specification of a Receptor signature.

Sets the requirements to meet for the java.lang.reflect.Method in order to be qualified as a receptor for the type of messages specified by the type of the envelopes <E>.

By extending this base class, descendants define the number of requirements:

  • the method annotation,
  • the specification of method parameters,
  • the set of allowed access modifiers,
  • the set of valid return types,
  • the set of allowed exceptions, that the method declares to throw (empty by default),
  • whether an ignored result, such as NoReaction, may be returned.
