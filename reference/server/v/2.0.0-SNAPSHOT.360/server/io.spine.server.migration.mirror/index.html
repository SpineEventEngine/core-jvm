class MirrorsMigrated : GeneratedMessage, MirrorsMigratedOrBuilder, ValidatableMessage
A number of `Mirror` projections, which were migrated
in a single request during the process of `MirrorMigration`.

The batch size for the migration is dictated by a `MirrorMigrationMonitor`. Hence, the value
of this message is usually equal to the batch size. Only the last request is expected to
return less or zero, indicating no records left to migrate.
