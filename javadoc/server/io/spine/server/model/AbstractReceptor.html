Package io.spine.server.model

Class AbstractReceptor

  • All Implemented Interfaces:
    io.spine.server.model.Receptor 
    @Immutable() 
public abstract class AbstractReceptor<T, M extends Message, C extends MessageClass<M>, E extends MessageEnvelope<out Object, out Signal<out Object, out Object, out Object>, out Object>, R extends MessageClass<out Object>>
 implements Receptor<T, C, E, R>

    An abstract base for wrappers over methods handling messages.

    Two message handlers are equivalent when they refer to the same method on the same object (not class).
