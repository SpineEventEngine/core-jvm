Package io.spine.server.route

This package contains classes and interfaces for obtaining entity identifiers.

See: Description

  • Annotation Summary 
    Annotation Description
    RouteAnnotates a static method of an Entity class for arranging message routing for this class of entities.
  • Interface Summary 
    Interface Description
    CommandRouteObtains an ID of an entity which would handle the command.
    EventRouteObtains a set of entity IDs for which to deliver an event.
    MulticastA route for a message to be delivered to several entities.
    RouteFnA common interface for routing functions for calculating identifiers of entities to which a message should be delivered taking the message and its context.
    StateUpdateRouteObtains a set of entity IDs for which to deliver an entity state update.
    UnicastA route for a message to be delivered to one entity.
  • Class Summary 
    Class Description
    CommandRoutingA routing schema for commands.
    DefaultCommandRouteObtains an ID of a command target entity from the first field of the command message.
    EventRoutingA routing schema for events.
    MessageRoutingAn abstract base for classes implementing routing schemas for routable messages such as commands, events, rejections, or entity states.
    MulticastRoutingAn abstract base for routing schemas that route a message to potentially more than one entity.
    StateUpdateRoutingA routing schema designed for delivering entity state updates.
