      • idClass

         static <I> Class<I> idClass(Class<out Entity<out Object, out Object>> cls)

        Obtains the class of identifiers from the passed entity class.

        Checks that this class of identifiers obtained from the passed entity class is supported by the framework.

        The type of entity identifiers (<I>) cannot be bound because it can be Long, String, Integer, and class implementing Message.

        We perform the check to detect possible programming error in declarations of entity and repository classes until we have compile-time model check.

      • stateClassOf

         static <S extends EntityState<out Object>> Class<S> stateClassOf(Class<out Entity<out Object, out Object>> entityClass)

        Retrieves the state class of the passed entity class.

        Though this method is public, it is not considered a part of the public API. It is used internally by other framework routines and not designed for efficient execution by Spine users.
