Package io.spine.server.entity.storage

Class EntityRecordStorage

  • All Implemented Interfaces:
    io.spine.server.Closeable , io.spine.server.storage.Storage , java.io.Closeable , java.lang.AutoCloseable 
    
public class EntityRecordStorage<I, S extends EntityState<I>>
extends RecordStorageDelegate<I, EntityRecord>

    A storage for EntityRecords.

    Delegates all storage operations to the underlying RecordStorage. For that, creates a new RecordStorage instance through the provided StorageFactory and configures it with the record specification corresponding to the stored EntityRecord.
