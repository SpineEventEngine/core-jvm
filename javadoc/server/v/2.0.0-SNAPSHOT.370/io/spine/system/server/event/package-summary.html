    An event emitted when a command is passed to a dispatcher after being acknowledged.
    Protobuf type spine.system.server.CommandDispatched

    A command was dispatched to a command handler.

This event is relevant only to histories of Aggregates and ProcessManagers.
    Protobuf type spine.system.server.CommandDispatchedToHandler

    An event emitted if the command cannot be handled due to a technical error.
    Protobuf type spine.system.server.CommandErrored

    An event emitted when a command is successfully handled.
    Protobuf type spine.system.server.CommandHandled

    An event emitted when a new command is received by the `CommandBus`.

    CommandRejectedKt
    An event emitted if the command cannot be handled due to a business rejection.
    Protobuf type spine.system.server.CommandRejected
    CommandScheduledKt
    An event emitted when the command is scheduled.
    Protobuf type spine.system.server.CommandScheduled
