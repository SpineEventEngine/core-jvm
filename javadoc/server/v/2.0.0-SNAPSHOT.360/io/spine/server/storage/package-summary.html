Package io.spine.server.storage

This package provides interfaces and base classes for storages.

See: Description

  • Interface Summary 
    Interface Description
    ExtractIdA method object to extract the value of a record identifier given an instance of a record.
    ColumnMappingThe value mapping rules of record columns.
    ColumnTypeMappingA persistence strategy of an entity column.
    StorageFactoryA factory for creating storages used by repositories, EventStore and io.spine.server.delivery.Delivery.
    StorageThe base interface for storages.
  • Class Summary 
    Class Description
    RecordSpecDefines the specification of a record in a storage.
    RecordWithColumnsA value of some message record along with the values of its columns.
    QueryConverterConverts the queries defined in Protobuf into the language of io.spine.query package.
    FieldMaskApplierA Function transforming the storage record by applying the given FieldMask to it.
    RecordStorageAn abstract base for storage implementations, which store the Protobuf messages as records.
    AbstractStorageAbstract base for storages.
    MessageStorageA storage which allows persisting the Messages as storage records.
    RecordStorageDelegateA RecordStorage which delegates all of its operations to another instance of RecordStorage.
    AbstractColumnMappingThe base implementation of mapping rules for the column values of a stored record.
    DefaultColumnMappingA default column mapping which stores all values "as-is".
