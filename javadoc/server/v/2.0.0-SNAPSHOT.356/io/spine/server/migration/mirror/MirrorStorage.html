Package io.spine.server.migration.mirror

Class MirrorStorage

  • All Implemented Interfaces:
    io.spine.server.Closeable , io.spine.server.storage.Storage , java.io.Closeable , java.lang.AutoCloseable 
    
public final class MirrorStorage
extends MessageStorage<MirrorId, Mirror>

    A contract for storages of legacy Mirror projections.

    Mirror was deprecated in Spine 2.0. The storage is intended to be used only in a scope of the migration.

    It exposes methods for performing read and write operations in batches.
