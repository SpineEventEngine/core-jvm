Package io.spine.server.storage

Class MessageStorage

  • All Implemented Interfaces:
    io.spine.server.Closeable , io.spine.server.storage.Storage , java.io.Closeable , java.lang.AutoCloseable 
    
public abstract class MessageStorage<I, M extends Message>
extends RecordStorageDelegate<I, M>

    A storage which allows persisting the Messages as storage records.

    The record specification is used to determine the record identifier and the columns.

    To persist the Entity data, see EntityRecordStorage, which uses not only the Entity state, but the Entity lifecycle attributes to prepare the storage record.
