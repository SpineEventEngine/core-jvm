public abstract class AbstractAssignee
extends AbstractCommandDispatcher implements Assignee

    The abstract base for non-aggregate classes that expose methods assigned to handle commands and post their results to EventBus.

    A command assignee is responsible for:

    • Changing the state of the business model in response to a command. This is done by one of the command-handling methods to which the assignee dispatches the command.
    • Producing corresponding events.
    • Posting events to EventBus.

    Event messages are returned as values from command-handling methods.

    A command assignee does not have its own state. So the state of the business model it changes is external to it. Even though such behaviour may be needed in some rare cases, using aggregates is a preferred way of handling commands.

    This class implements CommandDispatcher dispatching messages to methods declared in the derived classes.
