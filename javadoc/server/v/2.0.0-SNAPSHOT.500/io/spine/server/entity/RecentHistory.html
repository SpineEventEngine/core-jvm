Package io.spine.server.entity

Class RecentHistory

  • All Implemented Interfaces:
    
public abstract class RecentHistory<T extends Object, L extends HistoryLoader<?>>

    A view on the recent history of a TransactionalEntity, read lazily from a durable storage.

    The history serves its reads through a loader, installed by the repository managing the entity. The items are not cached on the entity side: an entity serves its signals and leaves the memory; caching, if any, belongs to the storage side.

    An entity created outside a repository has no loader installed, so its reads return no items.

    The kinds of recent histories are fixed by the framework: the constructor is internal.
