Package io.spine.server

Class GrpcContainer

  • All Implemented Interfaces:
    
public final class GrpcContainer

    Wrapping container for gRPC server.

    Maintains and deploys several of gRPC services within a single server.

    Uses ServerServiceDefinitions of each service.

    • Nested Class Summary

      Nested Classes 
      Modifier and Type Class Description
      public final class GrpcContainer.Builder

      The builder for GrpcContainer allows to define a port and services exposed by the container.
      public interface GrpcContainer.ConfigureServer

      Allows to configure the gRPC's Server instance, on top of which this GrpcContainer will operate.

      It is expected that the obtained builder of gRPC server is used to perform some fine-grained tuning of its features. The same instance of ServerBuilder should be returned.

      Example. 

      
val container =
    GrpcContainer.atPort(1654)
.withServer((server) -> server.maxInboundMessageSize(16_000_000))
                 // ...
                 .build();

      Please note this interface is a part of experimental API.
