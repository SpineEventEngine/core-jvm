public final class Quintet<A extends Message, B, C, D, E>
extends Tuple implements Element.AValue<A>, Element.BValue<B>, Element.CValue<C>, Element.DValue<D>, Element.EValue<E>

    A tuple of five elements.

    The first element must be a non-default Message and not Empty.

    Other four can be Message, Optional or Either.
