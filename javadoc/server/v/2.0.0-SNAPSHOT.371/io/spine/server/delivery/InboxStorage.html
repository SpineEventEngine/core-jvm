Package io.spine.server.delivery

Class InboxStorage

  • All Implemented Interfaces:
    io.spine.server.Closeable , io.spine.server.storage.Storage , java.io.Closeable , java.lang.AutoCloseable 
    @SPI() 
public class InboxStorage
extends MessageStorage<InboxMessageId, InboxMessage>

    A contract for storages of Inbox messages.

    The records a storage of this type are spreads across shards identified by a shard index.

    Typically, the storage instance is specific to the server environment and is used across BoundedContexts to store the delivered messages.

    • Constructor Detail

      • InboxStorage

        InboxStorage(StorageFactory factory, boolean multitenant)

        Creates a new instance of this storage.

        Generally, InboxStorage instances should be single-tenant only. It is so, because no distinction should be made for processing of InboxMessages, since it is batch-based anyway, and splitting batches even more (across tenants) reduces the performance.

        Parameters:
        factory - storage factory to create an underlying record storage
        multitenant - whether InboxStorage should be multi-tenant
