Package io.spine.server.storage

Class RecordSpec

  • All Implemented Interfaces:
    
public final class RecordSpec<I, R extends Message>

    Defines the specification of a record in a storage.

    Defines the identifier column and the collection of the data columns to store along with the message record for further querying. Each column defines a way to calculate the stored value basing on the passed message.

    In case an instance of EntityRecord is described by this spec, a special set of accessors for ID and Entity state columns is set. This is done via SpecScanner which perform analysis of Entity state, and creates getters for properties, considering that an instance of Any (being EntityRecord.state field) must be unpacked first.

    • Nested Class Summary

      Nested Classes 
      Modifier and Type Class Description
      public interface RecordSpec.ExtractId

      A method object to extract the value of a record identifier given an instance of a record.

      Once some storage is passed a record to store, the value of the record identifier has to be determined. To avoid passing the ID value for each record, one defines a way to obtain the identifier value from the record instance itself — by defining an ExtractId as a part of the record specification for the storage.
