Package io.spine.server.storage

Class AbstractColumnMapping

  • All Implemented Interfaces:
    io.spine.server.storage.ColumnMapping 
    
public abstract class AbstractColumnMapping<R>
 implements ColumnMapping<R>

    The base implementation of mapping rules for the column values of a stored record.

    Since record columns are proto-based and have a limited amount of possible types, this class allows descendants to override concrete type mapping rules in a convenient way.

    Some of the types are expected to be mapped in a way so that they support the ordering comparison operators ("greater than", "less than or equals", etc.). For details, see io.spine.client.Filters.

      • setupCustomMapping

         void setupCustomMapping(ImmutableMap.Builder<Class<out Object>, ColumnTypeMapping<out Object, out R>> builder)

        Allows to specify custom mapping rules.

        If some message types are needed to be stored differently to the generic messages, the rules for their storage can be specified using this method.

        The common examples of such messages are Timestamp and Version.

        The custom mapping can also be specified for marker interfaces of messages like io.spine.base.EventMessage.
