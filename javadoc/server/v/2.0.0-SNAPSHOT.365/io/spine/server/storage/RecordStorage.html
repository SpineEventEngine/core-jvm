Package io.spine.server.storage

Class RecordStorage

  • All Implemented Interfaces:
    io.spine.server.Closeable , io.spine.server.storage.Storage , java.io.Closeable , java.lang.AutoCloseable 
    
public abstract class RecordStorage<I, R extends Message>
extends AbstractStorage<I, R>

    An abstract base for storage implementations, which store the Protobuf messages as records.

    Each stored record must be identified.

    Additionally, some attributes may be stored along with the record itself to allow further querying.
