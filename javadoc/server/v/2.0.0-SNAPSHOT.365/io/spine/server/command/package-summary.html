Package io.spine.server.command

This package provides classes and interfaces related to command processing at the server side.

See: Description

  • Class Summary 
    Class Description
    AbstractAssigneeThe abstract base for non-aggregate classes that expose methods assigned to handle commands and post their results to EventBus.
    AssigneeEntityAn entity that has one or methods assigned to handle commands.
    AbstractCommandDispatcherThe abstract base for non-aggregate classes that dispatch commands to their methods and post resulting events to EventBus.
    AbstractCommanderThe abstract base for classes that generate commands in response to incoming messages.
    Builder
    A system command that implies no action.

Use this command type when a commanding method does not produce any output in response to
an event.

This type is treated by the framework in a special way. A `DoNothing` command is never posted
for processing after being emitted. Neither is it stored.

There can be no handlers for a `DoNothing` command.

Deprecated: please use `spine.server.command.DoNothing` instead.
