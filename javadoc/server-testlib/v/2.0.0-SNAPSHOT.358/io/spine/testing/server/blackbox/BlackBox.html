Package io.spine.testing.server.blackbox

Class BlackBox

  • All Implemented Interfaces:
    io.spine.logging.WithLogging , io.spine.server.Closeable , java.io.Closeable , java.lang.AutoCloseable 
    @VisibleForTesting() 
public abstract class BlackBox
 implements WithLogging, Closeable

    This class provides means for integration testing of Bounded Contexts.

    Such a test suite would send commands or events to the Bounded Context under the test, and then verify consequences of handling a command or an event.

      • close

         final void close()

        Closes the BlackBox performing all necessary clean-ups.

        This method performs the following:

        • Closes the tested BoundedContext.
        • Closes the associated Clients.

        Instead of a checked IOException, wraps any issues that may occur while closing, into an IllegalStateException.
