        Obtains the subject for the message at the given index.

        Extracts the message of type M from the outer object of type T at the specified index and returns a subject for asserting on it.

        Fails if the index is out of the range of the generated message sequence.

        Parameters:
        index - the zero-based index of the message to retrieve
        Returns:

        a subject for asserting on the unpacked message
