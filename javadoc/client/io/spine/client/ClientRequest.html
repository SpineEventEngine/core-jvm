Package io.spine.client

Class ClientRequest

  • All Implemented Interfaces:
    
public class ClientRequest
extends ClientRequestBase

    Entry point for creating client requests.

    An instance of this class is obtained via onBehalfOf or asGuest methods and then used for creating a specific client request e.g. for posting a command.

    A client request may be customized using fluent API provided by the classes derived from ClientRequestBase.

    Some features such as running an EntityQuery are available directly from this class.
